 PROJECT DESCRIPTION---------------

URL shortening is used to create shorter aliases for long URLs. These shortened aliases “short links.” Users are redirected to the original URL when they hit these short links. Short links save a lot of space when displayed, printed, messaged, or tweeted. Additionally, users are less likely to mistype shorter URLs.
